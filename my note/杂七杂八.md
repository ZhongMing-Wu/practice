##### 关于类中变量访问的一种情况

> 在一个类的方法中，如果参数是该类本身的一个实例，那么该实例可以直接访问其私有变量。

```java
class Person {
    private String name;

    public void setName(Person person) {
        name = person.name;  // 直接访问其私有变量
    }
}
```

##### 求一个十进制数对应二进制数中 1 的个数

```java
    public static int times2(int n){
        int count = 0;
        while(n!=0){
            count++;
            n = n&(n-1);
        }
        return count;
    }
```
